# DOK_mb_monitor

/wtf - получить данные о состоянии комплекса 

/get_msglog - получить msglog

/get_syslog - получить syslog

/get_csv - получить CSV с данными 

/get_data_in_3_hours - графики за последние 3 часа 

/get_data_in_12_hours - графики за последние 12 часов  

/get_data_in_24_hours - графики за последние 24 часа 

/get_data_in_3_days - графики за последние 3 дня  

/get_data_in_7_days - графики за последние 7 дней 

/get_data_in_14_days - графики за последние 14 дней 

/get_data_in_30_days - графики за последние 30 дней
